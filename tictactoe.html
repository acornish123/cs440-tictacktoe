Tic-Tac-Toe

Tic-Tac-Toe is not the most sophisticated game, but it has the nice property that the rules are simple, and the total number of game states (3^9) can be enumerated easily. Further, each game has a finite length. In this assignment you'll build a simple Tic-Tac-Toe class to assist with playing the game. Then we'll use the logic to make some calculations about the environment. A simple stub module has been provided.

Overview

  1. Download the stub tictactoe.py and open it in your favorite editor (emacs, right?!).
  2. Look through the code. I've written some functionality, and the goal is for you to complete it.
  3. Note that there are 5 methods/functions that are unimplemented, these are where you should add your code.
  4. Note further that the code assumes four spaces of indentation. Set your editor appropriately, this is the default for many.
  5. The code contains a main method with some argument parsing ability. You can get some details on usage with: python tictactoe.py --help

Implement the methods

Once the methods are implemented, you should be able to play the game of tic-tac-toe by running on the command line as follows:

>python tictactoe.py --play

Note that at the command line, you represent the state with {0,1,2} not {0,1,-1}. Eventhough the moves above are selected "randomly", the only remaining move makes 'X' win every time.
